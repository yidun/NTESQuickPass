## 1.Cocoapods 集成
#### 执行pod repo update更新。
#### Podfile 里面添加以下代码：
```ruby
# 以下两种版本选择方式示例

# 集成最新版SDK:
pod 'NTESQuickPass'

# 集成指定SDK，具体版本号可先执行 pod search NTESQuickPass，根据返回的版本信息自行决定:
pod 'NTESQuickPass', '~> 3.3.7'
```
### 保存并执行pod install即可，若未执行pod repo update，请执行pod install --repo-update
